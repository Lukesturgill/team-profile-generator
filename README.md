# Team Profile Generator

## Purpose
Team Profile Generator is a CLI application that allows the user to generate a HTML file containing their team members by filling in the prompts.

## Built With
* Node.JS
* CSS
* JavaScript
* Inquirer
* Jest

## Installation 
* `npm install`
* `node index.js`

## Test
* `npm run test`
